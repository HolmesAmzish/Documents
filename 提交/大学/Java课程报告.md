# 一、问题概述

聊天室共分为客户端和服务端两部分，服务器程序主要负责侦听客户端发来的消息，客户端需要登录到相应的服务器才可以实现正常的聊天功能。

服务器的主要功能：

1. 在特定端口上进行侦听，等待客户连接
2. 向已经连接服务器的客户发送系统消息
3. 统计在线人数
4. 当停止服务时，断开所有用户的连接

客户端的主要功能：

1. 连接到已经开启聊天服务的服务端
2. 用户可以配置连接后显示的用户名
3. 当服务器开启时。用户可以随时登陆和注销
4. 用户可以向所有人或一个人发送消息
